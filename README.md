# Investigating the Properties of Static Word Embeddings through the Lens of Semantic Field Theory
Chulpan Gazizullina

```analysis.ipynb``` основная тетрадка с кодом для анализа слов
папка ```data``` -- электронная версия словаря Шведовой, используемая в работе 
папка ```result``` -- подробные отчеты кластеризации на 3, 7 и 36 кластеров соответственно
файлы ```matrix``` представляют собой матрицу сопряженности код Шведовой| кластер
файлы ```words``` представляют собой список слов с указанными метками кластеров и кодами Шведовой
